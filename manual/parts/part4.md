Part IV documents the hardware abstraction layer and the eight microcontroller targets that plug into GoPLC over USB or Ethernet. The HAL chapter explains how GoPLC maps ST tags to real I/O pins on Raspberry Pi, PCF8574 I/O expanders, and ADC boards without requiring any Go code changes. The hardware target chapters — Parallax P2, Arduino, Teensy, Flipper Zero, RP2040-Zero, ESP32 HMI dongle, and Phidgets — each cover firmware upload, wire framing, and the full set of ST functions exposed by that target.
