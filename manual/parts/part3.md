Part III is the Structured Text programming reference. It opens with timers, counters, and the standard IEC 61131-3 function blocks, then documents every general-purpose library available to your ST code: math and string operations, data structures (arrays, maps, lists, queues, stacks, sets), JSON parsing and construction, multi-target logging and diagnostics, regular expressions, file and database I/O, HTTP client calls, CSV and INI parsing, cryptography and TLS, PLCopen motion control, resilience primitives (cache, circuit breaker, rate limiter), SMTP email, and the full OSCAT library for process-control math. If a function is built into GoPLC and callable from ST, its reference chapter lives here.
